# Inherit là gì?
- inherit không phải là attribute trong css mà nó chính là 1 value đặc biệt.
- Value này giúp ta thừa hưởng value cùng loại property từ thẻ cha trước đó.
- Nhưng nếu thẻ cha không có property cùng loại nó sẽ lấy value mặc định của property đó.

## Inherit property
- Có những loại thẻ ngay từ đầu đã sở hữu thuộc tính property mặc định rồi như thẻ &lt;strong&gt;
- vidu như các property của văn bản trong thẻ `color, font-family, font-size, line-height, and text-align`
