# Gradient ???
- Gradient trong css là quang phổ màu sắc, sự biến thiên của màu sắc.
- Gradient là 1 value của 1 property, hay dùng nhất chính là `background-image: linear-gradient();`
- Gradient có 3 dạng chính:
    1. Linear-Gradients: Quang Phổ tuyến tính.
    2. Radial-Gradients: Quang Phổ xuyên tâm.
    3. Conic-Gradients: Quang Phổ xoắn ốc.

# Repeating gradient?
- repeating là 1 function đúng như tên gọi nó sẽ lặp lại dải màu nếu như quang phổ đó chưa chiếm hết không gian 100%.
