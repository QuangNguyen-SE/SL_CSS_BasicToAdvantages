## What is pseudo-class?
- Pseudo-class is a pre-written class in CSS
- Allow to **style one or many elements/nodes in DOM tree** base on **user interaction without Javascript**
## How many specific group type of Pseudo-class?
1. Interactive class
    - User's interaction 
2. Links class
    - Link tag (&lt;a&gt;)
3. Advance class
    - Advance user's interaction
4. Structured class
    - Base on position of tag (Parent-Child tag)
5. Logic
    - High advance class
    - Clear code
6. Lang & Display
    - Define language of webpages
    - Responsive display
